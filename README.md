**CI/CD LAB – Duplicate Removal Pipeline**

**Project Overview**

This project demonstrates the implementation of a simple CI/CD pipeline for automated duplicate removal using Python, GitHub, and GitHub Actions. The workflow reflects Agile Data Science practices by automating data preparation, testing, and validation processes.

The project was developed using Google Colab and integrated with GitHub Actions to enable continuous integration whenever changes are pushed to the repository.

**Objectives**

--Automate duplicate row removal from a dataset

--Implement automated testing using pytest

--Configure a CI/CD pipeline using GitHub Actions

--Demonstrate Agile Data Science workflow practices

--Enable rapid feedback through continuous integration

**Dataset Description**

The dataset used in this project is a sample CSV dataset containing intentionally duplicated records for testing purposes. The duplicate removal script processes the dataset and generates a cleaned output file with duplicates removed.

**Files generated:**

dataset.csv → Original dataset
processed_dataset.csv → Cleaned dataset
test_output.csv → Output generated during automated testing
